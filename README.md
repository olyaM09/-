# Expense Tracker — Трекер расходов

**Автор:** Шумкина Ольга Андреевна

## Описание

Приложение для учёта личных расходов с графическим интерфейсом (tkinter).  
Позволяет добавлять расходы с категорией и датой, фильтровать по категории/дате, считать сумму за произвольный период, сохранять данные в JSON.

## Как использовать

1. Установите Python 3.7+.
2. Запустите файл `expense_tracker.py`:
   ```bash
   python expense_tracker.py

## Пошаговая инструкция использования Expense Tracker

Первый запуск

1. Сохраните код в файл expense_tracker.py
2. Запустите программу:
   · Откройте командную строку/терминал в папке с файлом
   · Введите: python expense_tracker.py
   · Откроется окно приложения

Шаг 1: Добавление расхода

1. Введите сумму (например: 350.50)
2. Выберите категорию из выпадающего списка (Еда, Транспорт, Развлечения, Здоровье, Другое)
3. Введите дату в формате ГГГГ-ММ-ДД (по умолчанию стоит сегодняшняя дата)
4. Нажмите кнопку «Добавить расход»
5. Запись появится в таблице внизу

Пример:

```
Сумма: 250.00
Категория: Еда
Дата: 2026-05-05
→ Нажать «Добавить расход»
```


Шаг 2: Просмотр всех расходов

· Все добавленные записи отображаются в таблице с колонками:
  · ID (номер записи)
  · Сумма
  · Категория
  · Дата


Шаг 3: Фильтрация расходов

Фильтр по категории:

1. В блоке «Фильтр по категории» выберите нужную категорию (например, «Транспорт»)
2. Нажмите «Применить фильтр»
3. В таблице останутся только расходы выбранной категории

Фильтр по дате:

1. В поле «Дата от» введите начальную дату (например, 2026-05-01)
2. В поле «до» введите конечную дату (например, 2026-05-31)
3. Нажмите «Применить фильтр»
4. Увидете расходы только за этот период

Комбинированный фильтр:

· Можно фильтровать одновременно по категории И по диапазону дат

Сброс фильтра:

· Нажмите «Сбросить фильтр» — покажутся все расходы


Шаг 4: Подсчёт суммы за период

1. В нижней части окна найдите блок «Период (от и до)»
2. Введите начальную дату (например, 2026-05-01)
3. Введите конечную дату (например, 2026-05-31)
4. Нажмите «Подсчитать сумму»
5. Справа отобразится общая сумма расходов за выбранный период

Пример:

```
Период от: 2026-05-01
Период до: 2026-05-07
→ Нажать «Подсчитать сумму»
→ Результат: "Сумма: 1250.75"
```


Шаг 5: Ошибки при вводе (что программа проверяет)

Программа не даст добавить расход, если:

× Сумма отрицательная или ноль (например, -100 или 0)
→ Ошибка: «Сумма должна быть положительным числом»

× Сумма не число (например, abc)
→ Ошибка: «Сумма должна быть положительным числом»

× Дата в неправильном формате (например, 05.05.2026 или 2026-13-45)
→ Ошибка: «Неверный формат даты. Используйте ГГГГ-ММ-ДД»

× Пустые поля (не заполнена сумма, категория или дата)
→ Ошибка: «Все поля обязательны для заполнения»


Шаг 6: Сохранение данных

· Все расходы автоматически сохраняются в файл expenses.json
· При следующем запуске программы данные загрузятся автоматически
· Файл expenses.json создаётся в той же папке, где находится программа
